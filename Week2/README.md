# These are the activities of week 2

## 1. Activity 1

1. Function cacl_sum: input two integer values and calculate their sum.

2. Function calc_power: input two integer values and calculate their power.

3. Function calc_complex: calculate the sum of two complex numbers.

## 2. Activity 2

1. Define a student_info class

2. Define get_info method to get the student info of class

3. Define display method to print the student info of class

4. In main function, create student_info class and call its method to get student info and display.

## 3. Activity 3

1. Define Calculator class to replace three functions of activity 1.

2. Define get_input function to get two integer numbers and create Calculator object, then call its set_data to assign two integer numbers as member variables.

3. Define output function to call functions Calculator object and output the result.

4. Define main function to call get_input and output functions.
